glass

Glass is a java library providing a web ui for quartz scheduling library.

Quartz is the standard scheduler for java but it lacks of a pleasant web ui.

10 minutes test

A maven 2 environment is required for the test.

First thing to do is to checkout source code from git

git clone git://github.com/dbourdette/glass.git

Then launch tomcat with maven

mvn tomcat:run -P test

A glass instance should be running on http://localhost:8080
